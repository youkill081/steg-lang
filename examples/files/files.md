# Examples Fichiers

Plusieurs exemples ici sont présents pour montrer la capacité de gestion des fichiers

## Readfile

[readfile.stegasm](readfile/readfile.stegasm) Ouvre un fichier dynamiquement sur le PC et affiche son contenu dans la console.  

Vous pouvez le lancer avec l'image [readfile.png](readfile.png):   
````shell
./stegnocode ./readfile.png
````

Output:  
````
Hello, je suis un exemple qui peux etre lus avec readfile.stegasm !
````

## Writefile

[writefile.stegasm](writefile/writefile.stegasm) Crée un fichier sur votre machine avec une string écrite à l'intérieur.  

Vous pouvez le lancer avec l'image [writefile.png](writefile.png):
````shell
./stegnocode ./writefile.png
````
